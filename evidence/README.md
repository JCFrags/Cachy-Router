# Evidence Notes

The public repository keeps small, curated result summaries only when they are
sanitized and useful for readers. Raw runtime artifacts stay out of Git.

Do not commit:

- slot files or cache blobs;
- raw prompts, raw requests, raw responses, or transcripts;
- runtime logs, PID files, auth token files, or environment captures;
- local deployment inventories with private LAN topology;
- model files or generated package archives.

Historical lab evidence from the original research workspace should be converted
to a small redacted summary before it is copied into this standalone project.
